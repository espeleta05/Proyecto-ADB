# Proyecto-ADB
Proyecto final de bases de datos avanzadas     Sistema digital de control y seguimiento de vacunación infantil

Este proyecto se centra en el control del esquema de vacunación infantil. El sistema permitirá registrar niños, tutores, vacunas, esquemas oficiales, aplicaciones y responsables. Mediante NFC, cada niño podrá ser identificado rápidamente para consultar su historial.
Se deberán implementar validaciones por edad, intervalos entre dosis y esquemas incompletos, así como reportes oficiales y análisis poblacionales.
Objetivo: Desarrollar un sistema que garantice el seguimiento adecuado del esquema de vacunación infantil, asegurando integridad histórica, validaciones clínicas y generación de reportes confiables para instituciones de salud.
